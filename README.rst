plonetheme.emgame2012
=====================

This is the theme for the emgame 2012 website.



Usage
-----

- Add ``plonetheme.emgame2012`` to your buildout configuration:

::

    [instance]
    eggs +=
        plonetheme.emgame2012

- Install the generic import profile.
